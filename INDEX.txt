================================================================================
                    SHTF OFFLINE SURVIVAL RESOURCE KIT
                         Last updated: 2026-02-19
================================================================================

This folder contains offline resources for when infrastructure goes down.
Everything here works WITHOUT internet access. Total size: ~230GB+

================================================================================
QUICK START
================================================================================

1. WIKIPEDIA & ENCYCLOPEDIAS
   Open Kiwix app (in /Applications) > File > Open > navigate to wikipedia/ folder
   OR run: ./tools-scripts/launch-wikipedia.sh

2. AI ASSISTANT (LOCAL)
   ollama run llama3.2:3b
   (or any other model - see tools-scripts/pull-all-models.sh)

3. MAPS
   Install QGIS (brew install --cask qgis) to view .osm.pbf files
   Or use Organic Maps app (App Store) for phone-friendly offline maps
   USGS Topo maps in maps/topo/pdfs/ - open as GeoPDFs

4. MEDICAL REFERENCE
   Open medical/ folder - PDFs and ZIM files for Kiwix

================================================================================
HOW TO USE EACH RESOURCE
================================================================================

WIKIPEDIA OFFLINE:
  1. Open Kiwix.app (already installed in /Applications)
  2. File > Open ZIM File
  3. Browse to the wikipedia/ folder in this kit and open any .zim file
  4. Search and browse just like online Wikipedia
  5. All Stack Exchange and DevDocs ZIMs also open in Kiwix

LOCAL AI:
  1. Open Terminal
  2. Run: ollama run llama3.2:3b
  3. Chat with the AI - works completely offline
  4. For coding help: ollama run qwen2.5-coder:3b
  5. Ask it anything - medical advice, how-to, calculations, etc.

MAPS:
  For Mac: brew install --cask qgis, then open .osm.pbf files
  For phone: Download Organic Maps from App Store, it uses OpenStreetMap
  For best offline phone maps: Download regions in Organic Maps while online
  USGS Topo Maps: Open GeoPDF files in Preview or any PDF viewer
    - These show elevation contours, trails, water features, and terrain

MEDICAL:
  Open PDFs directly, or use Kiwix for the MDWiki medical encyclopedia
  Priority reading: Where There Is No Doctor (comprehensive village healthcare)

RADIO:
  Program your Baofeng UV-5R using the programming guide
  Key emergency frequencies:
    - 146.520 MHz (2m calling frequency)
    - 446.000 MHz (70cm calling frequency)
    - 156.800 MHz (Marine Channel 16 - distress)
    - 462.5625 MHz (FRS Channel 1)
    - NOAA Weather: 162.400, 162.425, 162.450, 162.475, 162.500, 162.525, 162.550

FOOD & WATER:
  Canning: USDA Complete Guide to Home Canning is the gold standard
  Foraging: Washington State guide has photos for PNW plant ID
  Seeds: Seed Saving Guide covers how to save & store seeds for next season
  Preservation: Dehydration, smoking, and raw food preservation guides

SOLAR/POWER:
  Start with the NREL off-grid guide for system sizing
  Video guides walk through complete DIY solar installations
  Solar Electric System Design for technical specifications

SANITATION:
  WHO WASH guidelines for emergency water/sanitation
  Emergency Toilet Guidebook for building toilets without plumbing
  CDC hygiene guidelines for disease prevention

STACK OVERFLOW & STACK EXCHANGE:
  Open any .zim file in Kiwix to search Q&A offline
  Stack Overflow alone has 75GB of programming knowledge
  Specialized sites: electronics, mechanics, ham radio, gardening, etc.

DEVELOPER DOCS:
  19 programming language/tool references in reference/devdocs/
  Open in Kiwix - covers Python, Rust, Go, C/C++, JavaScript, SQL, etc.
